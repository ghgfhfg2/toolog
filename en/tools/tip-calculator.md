## Fast tip and bill split in one place
Use this tool to calculate:
- Tip amount
- Final total to pay
- Per-person split amount

Great for travel, group dinners, and quick settlement chats.

## Key features
- Tip by percentage or fixed amount
- Include or exclude tax/service fees from tip base
- Split total by number of people
- Instant updates and copy-ready summary

## How to use
1. Enter base amount and optional tax/service fees.
2. Choose tip mode (percentage or fixed amount).
3. Set people count for split bill.
4. Review tip, total, and per-person amount.
