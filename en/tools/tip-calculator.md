## Fast tip and bill split in one place
When traveling or dining in groups, tip math can be surprisingly annoying.
This tool calculates all key values together:
- Tip amount
- Final amount to pay
- Per-person split

## Key features
- **Two tip modes**: percentage or fixed amount
- **Flexible tip base**: subtotal only or include tax/service fees
- **Automatic split** by number of people
- **Real-time updates** with copy-ready result summary

## How to use
1. Enter subtotal, and optionally tax/service fee.
2. Select tip mode (percentage or fixed amount).
3. Set people count for splitting.
4. Check tip, total, and per-person amount.

## FAQ
### Can this be used for non-restaurant payments?
Yes. Any payment scenario with optional extra amount and split is supported.

### Does the tool force a specific currency?
No. It is currency-agnostic and works as a practical calculator.

### Is this an official billing tool?
No. It is for quick estimation and settlement support.
