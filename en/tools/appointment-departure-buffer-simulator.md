## Why use it?
Many late arrivals come from missing small delays, not from travel time alone.
This simulator helps you plan:
- when to start getting ready
- when to leave home
- how much extra buffer to add
- how risky the current plan looks

## How to use it
1. Enter the appointment date and time.
2. Add travel time and prep time.
3. Pick a buffer style.
4. Check risk factors like transfers, bad weather, unfamiliar locations, or extra items.
5. Review the recommended start and leave times.
