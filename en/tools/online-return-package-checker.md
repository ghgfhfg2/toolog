## Why use this checker?
Online returns often fail because of small missed steps: a deadline passes, a component is missing, or there is no photo evidence for a defect. This checker turns the return reason, deadline, packaging state, and pickup method into a practical checklist.

## How to use
1. Enter the item and components, one per line.
2. Choose the return reason and days left.
3. Select package state, return method, and whether to include photo evidence.
4. Press `Make checklist`.
5. Copy the result before submitting the return request.

## Summary
The Online Return Package Checker is a checker-style tool for preparing return photos, components, packaging, and pickup steps before the actual store return process.
