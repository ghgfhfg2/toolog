# Subscription Cleanup Priority Simulator
Use this tool to rank subscriptions into keep, review/pause, and cancel-first groups.

## When it helps
- You want to reduce fixed monthly costs
- Several billing dates are coming soon
- Some tools are essential, but others are barely used

## How to use it
1. Enter up to five subscriptions.
2. Set monthly cost, usage frequency, replacement availability, and next billing window.
3. Mark shared plans or work/study essentials.
4. Review the recommended order.

## Notes
This tool runs in your browser and does not store your subscription list on the server.
