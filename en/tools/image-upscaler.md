## Why use it?
Directly enlarging a small image often makes edges blurry. This tool scales first and applies light sharpening so results look clearer.

## How to use
1. Upload an image.
2. Select 2x/3x/4x scale.
3. Run upscale and download the result.

## Tip
Start with 2x for natural-looking output, then test 3x/4x only if needed.
