## Why use the List Format Converter?
Plain line lists often need to become spreadsheet rows, Markdown tables, JSON arrays, or numbered lists. This tool removes the small repetitive formatting work.

## How to use it
1. Paste one item per line.
2. Choose CSV, Markdown table, JSON array, or numbered list.
3. Adjust cleanup options.
4. Convert and copy the result.
