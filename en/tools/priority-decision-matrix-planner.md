## Why use a Priority Decision Matrix Planner?
When several tasks feel urgent at once, **deciding what to start first can waste more time than the work itself**.
This tool helps you compare urgency, impact, effort, and confidence so you can quickly sort what to do now, what to handle this week, and what to hold for later.

## How to use it
1. Add up to five tasks or options.
2. Rate urgency, impact, effort, and confidence for each one.
3. Review the ranked result with action labels like `Do now`, `This week`, or `Hold`.
4. Copy the result into your notes or task manager.

## Good use cases
- Reordering today’s overloaded task list
- Explaining team priority choices more clearly
- Comparing which idea is worth testing first
