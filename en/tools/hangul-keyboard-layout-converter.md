## Why use this tool?
If you have ever typed a full word or sentence and then noticed the keyboard layout was wrong, you already know how annoying it is.
This tool quickly recovers common Korean/English keyboard-layout mistakes without making you retype everything.

## How it works
1. Paste the mistyped text.
2. Choose auto detect or a manual direction.
3. Review the recovered result.
4. Copy and paste it where you actually need it.

## Useful when
- you typed Korean while the keyboard was in English mode
- you typed English while the keyboard was in Korean mode
- you want to fix chat messages, search queries, titles, or notes quickly
