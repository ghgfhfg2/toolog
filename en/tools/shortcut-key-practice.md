## Why use the Shortcut Key Practice Quiz?
Keyboard shortcuts can save a surprising amount of time, but they are easy to forget when your hand reaches for the mouse first. Browser tab control, text editing, and window switching stick better when you practice them briefly and repeatedly.

This learning-type tool lets you choose an operating system and practice area, then answer quiz questions by matching an action to the right shortcut. Missed questions stay in a review list so you can check memory instead of only reading a static list.

## How to use it
1. Choose Mac, Windows, or common shortcuts.
2. Pick browser, editing, system, or all categories.
3. Press `New question` and read the action prompt.
4. Choose the matching shortcut and read the explanation.
5. Use `Review missed` to retry the shortcuts you got wrong.

## Especially useful when
### 1) A three-minute warm-up before work
A few questions before opening mail or documents can bring useful shortcuts back to mind.

### 2) Switching between Mac and Windows
Practice only the environment you need when Ctrl, Command, Option, and Alt combinations become confusing.

### 3) Team onboarding or training
A quiz checks whether people remember shortcuts better than simply handing them a list.

## FAQ
### What if shortcuts differ by app?
This tool focuses on common browser and general editing shortcuts. For app-specific shortcuts, check that app’s help page as well.

### Why not capture real key presses?
Real shortcut capture in a browser can conflict with actions like opening a new tab or navigating pages. A multiple-choice quiz is safer.

### Can I review only missed questions?
Yes. Missed questions stay in the current session and can be retried with the review button. They reset when the page is refreshed.
