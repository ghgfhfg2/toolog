# Search Console 빠른 등록 가이드

## 1) 사이트 등록
1. Google Search Console 접속
2. 속성 추가 → URL 접두어에 `https://toolog.sooyadev.com` 입력
3. 소유권 확인(권장: DNS TXT)

## 2) 사이트맵 제출
- 제출 URL: `https://toolog.sooyadev.com/sitemap.xml`
- 상태가 `성공`인지 확인

## 3) 초기 점검(발행 후 1~3일)
- 색인 생성 여부 확인
- 색인되지 않음 원인(크롤링/중복/리디렉션) 점검
- 상위 쿼리와 클릭률(CTR) 확인

## 4) 운영 루틴(주 1회)
- CTR 낮은 페이지: 제목/메타 설명 A/B 수정
- 노출 대비 클릭 낮은 페이지: 첫 문단/FAQ 개선
- 성능 저하 페이지: 이미지 최적화 재점검
