# Search Console 빠른 등록 가이드

## 1) 사이트 등록
1. Google Search Console 접속
2. 속성 추가 → URL 접두어에 `https://toolog.sooyadev.com` 입력
3. 소유권 확인(권장: DNS TXT)

## 2) 사이트맵 제출
- 제출 URL: `https://toolog.sooyadev.com/sitemap.xml`
- 상태가 `성공`인지 확인

## 3) 실적 진단 기준
GSC 실적은 클릭보다 먼저 노출과 평균 순위를 봅니다.

- 노출이 낮음: 구글이 페이지를 충분히 보여주지 않는 상태입니다. 검색어 수요, 제목/H1, 본문 답변성, 내부 링크, 주제 클러스터를 먼저 점검합니다.
- 평균 순위 8~20위: 조금만 개선해도 1페이지 진입 가능성이 있는 기회 구간입니다. 우선 수정 후보로 봅니다.
- 노출은 있는데 CTR이 낮음: 제목과 메타 설명이 검색 의도와 덜 맞는 상태일 수 있습니다.
- 클릭은 있는데 순위가 흔들림: 본문 깊이, FAQ, 관련 도구 링크, 페이지 경험을 보강합니다.

## 4) 주간 운영 루틴
1. `검색결과 > 검색어`에서 최근 28일 기준 노출 상위 20개를 확인합니다.
2. `검색결과 > 페이지`에서 노출은 있으나 평균 순위가 8~20위인 페이지를 고릅니다.
3. 후보 페이지마다 실제 검색어 표현을 제목, H1, 첫 문단, 예시, FAQ에 반영합니다.
4. 홈/카테고리/관련 도구에서 해당 페이지로 내부 링크를 추가하거나 앵커 문구를 조정합니다.
5. 배포 후 색인 상태를 확인하고, 7~14일 뒤 같은 검색어의 노출·순위·CTR 변화를 비교합니다.

## 5) 우선 개선 후보 예시
- 이미지: `이미지 용량 줄이기`, `PNG 압축`, `이미지 크기 조절`, `WebP 변환`
- 텍스트: `글자수 계산기`, `블로그 금칙어`, `대소문자 변환`
- 계산: `할인 계산기`, `대출 이자 계산기`, `복리 계산기`, `예금 이자 계산기`

새 툴 추가보다 위 후보 중 GSC에서 이미 노출이 잡힌 페이지를 먼저 강화합니다.
