## このランチメニューピッカーが役立つ理由
食べること自体よりも、「何を食べるか」を決めるほうが意外と大変なことがあります。
特にお腹が空いているときほど、こうした場面で時間を使いがちです。

- ひとりで手早く食べて済ませたいとき
- チームランチなのに全員が「何でもいい」と言うとき
- 予算はあるのにメニューが思い浮かばないとき
- 汁ものやご飯ものなど、今ほしい条件がある程度はっきりしているとき

このツールは、そうした迷いを減らすために作られました。
**食事シーン + 一緒に食べる人数感 + 予算 + 今の気分**を選ぶだけで、共有しやすい形で候補を3つすぐに絞り込みます。

## どうやっておすすめを出すのか
とても大きなAIで判断するのではなく、実際の食事選びで効く条件を軽く反映する方式です。

1. **食事時間**: ランチ / 夕食 / 夜食
2. **人数の性格**: ひとり / 2〜3人 / チームランチ
3. **予算帯**: 軽め / 普通 / 余裕あり
4. **気分**: しっかり一食 / 軽くすっきり / 辛くて刺激的 / 手早く食べたい / 会話しやすい
5. **希望オプション**: 温かいメニューを含める、ご飯ものを含める

条件に合うメニューほど点数を高くし、上位候補を表示します。
つまり、ただの無作為な抽選ではなく、**今の状況で選びやすいメニューに整理してくれる選択補助ツール**に近いです。

## 特に使いやすい場面
### 1) 会社のランチメニューを決めたいとき
チームランチでは、料理そのものよりも**決める疲れ**が大きいことがあります。
結果をそのままグループチャットに貼れば、やり取りの時間を減らせます。

### 2) 一人で食べるけれど長く悩みたくないとき
予算と気分を決めれば、一人で食べやすい候補が上に来ます。
短い昼休みに特に使いやすいです。

### 3) 夕食が決まらないとき
ランチだけでなく夕食や夜食にも対応しているので、仕事のあとに「お腹は空いたけど何を食べるか決まらない」というときにもすぐ使えます。

## 使い方
1. 食事シーンを選びます。
2. 一緒に食べる人のタイプを選びます。
3. 予算を選びます。
4. 今の気分を選びます。
5. 温かいメニューやご飯ものを含めるか選びます。
6. メニュー推薦ボタンを押すと、上位3候補が生成されます。
7. 結果をメモやメッセンジャーにそのまま貼り付けられます。

## 使用例
### 例1) 一人で手早いランチ
- 食事シーン: ランチ
- 一緒に食べる人: ひとり
- 予算: 軽め
- 気分: 手早く食べたい

→ ハンバーガー、弁当・丼、軽食セットのような早く食べやすい候補が上に出やすくなります。

### 例2) 会話しやすいチームランチ
- 食事シーン: ランチ
- 一緒に食べる人: チームランチ / 複数人
- 予算: 普通
- 気分: 会話しやすい

→ 寿司、パスタ、マーラータンのように、一緒に食べながら話しやすいメニューが優先されます。

### 例3) 夕食にしっかり温かいものを食べたいとき
- 食事シーン: 夕食
- 一緒に食べる人: 2〜3人
- 予算: 普通
- 気分: しっかり一食
- 温かいメニューを含める: オン

→ クッパ、キムチチゲ、カルグクス、フォーのような温かく満足感のある候補が合いやすくなります。

## FAQ
### 候補数が少なく見えることはありますか？
条件がかなり狭いと候補プールが小さくなることがあります。その場合は、一部条件を少し緩めて再提案します。

### ダイエット食やベジタリアンのような詳細フィルターはありますか？
現在は、日常の素早い選択に集中した軽量版です。今後、より細かい食事フィルターを追加する余地はあります。

### チームランチでそのまま投票のように使えますか？
はい。3つの候補をそのままチャットに貼って、すぐ選ぶ使い方に向いています。

完全ランダムより少し現実的な提案がほしいときに向いています。
