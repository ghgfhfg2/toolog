# サブスク整理優先度シミュレーター
月額サービスを、維持・保留検討・解約優先の順で見直したい時に使えるツールです。

## こんな時に便利です
- 固定費を見直したい
- 次回決済が近いサービスを先に確認したい
- 仕事で必要なものと、あまり使っていないものを分けたい

## 使い方
1. 最大5件まで入力します。
2. 月額料金、利用頻度、代替手段、次回決済時期を設定します。
3. 共有プランか、仕事・学習に必須かをチェックします。
4. 優先順位を確認します。

## メモ
このツールはブラウザ内で動作し、入力内容をサーバーに保存しません。
