## 중고거래 준비 체크리스트 플래너가 왜 필요할까요?
중고거래는 사진만 올리면 끝날 것 같지만, 실제로는 **상태 확인, 구성품 점검, 약속 메시지, 포장, 입금·전달 준비**까지 신경 쓸 것이 꽤 많습니다.
특히 퇴근 후 급하게 거래를 잡거나 여러 물건을 동시에 올릴 때는 빠뜨리는 항목이 생기기 쉽습니다.

이럴 때 자주 생기는 문제가 있습니다.
- 충전기, 박스, 설명서처럼 같이 줘야 할 구성품을 놓치는 경우
- 직거래 장소와 시간은 정했는데 테스트 준비를 못 한 경우
- 택배거래인데 완충재, 송장 정보, 계좌 확인을 늦게 챙기는 경우
- 사용 흔적이나 하자를 설명문에 반영하지 못해 다시 문의가 오는 경우
- 거래 직전 개인 연락처나 주소 노출을 충분히 점검하지 못하는 경우

이 도구는 그런 실수를 줄이기 위해 만들었습니다.
**물품 종류 + 거래 방식 + 테스트 가능 여부 + 포장 조건**을 고르면, 판매 전 확인 항목과 거래 직전 준비 목록을 단계별로 바로 정리해줍니다.

## 어떻게 동작하나요?
사용 방식은 단순합니다.

1. 판매할 물품 종류를 고릅니다.
2. 직거래인지 택배거래인지 거래 방식을 선택합니다.
3. 전원 테스트 가능 여부, 구성품 포함 여부, 포장 필요 여부를 체크합니다.
4. 고가 물품인지, 하자 고지 메모를 따로 넣을지 선택합니다.
5. `체크리스트 생성`을 누르면 판매 전 점검, 약속 직전 준비, 거래 후 확인 항목이 생성됩니다.

결과에는 보통 아래 정보가 함께 정리됩니다.
- **판매 전 확인할 것**
- **약속 직전 / 발송 직전 준비할 것**
- **거래 후 남길 메모**
- **이번 거래에서 특히 신경 쓸 포인트**

## 이런 상황에서 특히 유용해요
### 1) 당근마켓, 번개장터, 중고나라에 막 올리기 전
사진은 찍었는데 설명문에 무엇을 더 써야 할지 막막할 때, 기본 점검 목록을 빠르게 만들 수 있습니다.

### 2) 직거래 약속을 잡았는데 준비 시간이 많지 않을 때
테스트, 현금영수증 여부, 거스름돈, 충전 상태, 만날 장소 메모처럼 놓치기 쉬운 항목을 한 번에 정리할 수 있습니다.

### 3) 택배거래 발송 직전 확인이 필요할 때
완충 포장, 송장용 정보, 계좌 확인, 발송 후 안내 메시지까지 흐름을 잡는 데 도움이 됩니다.

## 사용 방법
1. 물품 종류와 거래 방식을 고릅니다.
2. 전원 테스트, 구성품 포함, 포장 필요 여부를 체크합니다.
3. 고가 물품 여부와 하자 고지 메모 포함 여부를 선택합니다.
4. `체크리스트 생성` 버튼을 누릅니다.
5. 결과를 복사해 메모앱, 판매글 초안, 거래 대화에 맞게 활용합니다.

## 사용 예시
### 예시 1) 휴대폰 직거래
- 물품 종류: 전자기기
- 거래 방식: 직거래
- 테스트 가능: 가능
- 구성품 포함: 충전 케이블, 박스 있음

→ 전원 확인, 초기화 여부, 계정 로그아웃, 액정/카메라 상태, 현장 테스트 준비 같은 항목이 강조됩니다.

### 예시 2) 의류 택배거래
- 물품 종류: 의류/잡화
- 거래 방식: 택배거래
- 포장 필요: 높음
- 하자 고지 메모: 포함

→ 오염/올풀림 확인, 실측 정리, 접는 방식, 방수 포장, 발송 후 송장 전달 항목이 잘 맞습니다.

### 예시 3) 가구·생활용품 직거래
- 물품 종류: 가구/생활용품
- 거래 방식: 직거래
- 고가 물품: 체크
- 테스트 가능: 일부 가능

→ 부품 누락 여부, 분해 가능 여부, 이동 동선, 차량 적재 확인, 거래 장소 안전성 점검이 우선으로 나옵니다.

## FAQ
### 직거래 장소 추천도 해주나요?
현재 버전은 장소 자체를 추천하기보다, 거래 방식에 맞는 준비 항목을 정리하는 데 초점을 둡니다. 다만 고가 물품 직거래일수록 사람이 많은 공공장소와 테스트 가능한 장소를 우선 고려하는 것이 좋습니다.

### 전자기기 말고도 쓸 수 있나요?
네. 의류, 도서, 가구·생활용품처럼 자주 거래하는 범주를 기준으로 체크리스트가 달라지도록 만들었습니다.

### 결과를 그대로 판매글에 붙여도 되나요?
체크리스트는 판매자용 준비 메모에 가깝습니다. 판매글에는 핵심 상태, 구성품, 거래 방식, 하자 여부만 간결하게 정리해 옮기는 것을 권합니다.

중고거래 준비 체크리스트 플래너는 **판매 전 확인사항, 약속 직전 준비, 거래 후 메모까지 단계별로 정리해주는 planner형 도구**입니다.
직거래와 택배거래 모두에서 빠뜨리기 쉬운 준비를 줄이고, 더 매끄럽게 거래를 진행하고 싶을 때 특히 유용합니다.
