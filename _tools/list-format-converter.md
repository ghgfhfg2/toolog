## 왜 목록 형식 변환기가 필요할까요?
메신저나 메모장에 적어둔 줄글 목록을 업무 문서, 블로그 글, 스프레드시트, 개발 문서에 붙여넣으려면 형식을 다시 맞추는 시간이 은근히 듭니다. 항목마다 번호를 붙이고, CSV로 쉼표를 이스케이프하고, 마크다운 표의 구분선을 넣는 작업은 단순하지만 반복되기 쉽습니다.

이 도구는 한 줄에 하나씩 적힌 목록을 **CSV, 마크다운 표, JSON 배열, 번호 목록**으로 바로 바꿔줍니다. 브라우저 안에서만 처리되므로 간단한 업무 목록, 회의 안건, 블로그 소재, 체크리스트 초안을 빠르게 정리할 때 좋습니다.

## 어떻게 쓰나요?
1. 변환할 목록을 한 줄에 하나씩 붙여넣습니다.
2. 결과 형식을 CSV, 마크다운 표, JSON 배열, 번호 목록 중에서 고릅니다.
3. 필요하면 빈 줄 제거, 앞뒤 공백 제거, 중복 항목 제거 옵션을 켭니다.
4. `목록 변환`을 누르고 결과를 확인합니다.
5. 복사 버튼으로 문서, 스프레드시트, 노션, README 등에 붙여넣습니다.

## 이런 상황에서 특히 유용해요
### 1) 메모 목록을 스프레드시트에 넣을 때
줄글 목록을 CSV로 바꿔 복사하면 스프레드시트나 표 편집기에 붙여넣기 쉽습니다.

### 2) README나 블로그에 표를 만들 때
항목 목록을 마크다운 표로 바꾸면 구분선과 행 형식을 매번 손으로 만들지 않아도 됩니다.

### 3) 개발 문서나 자동화 입력값을 만들 때
목록을 JSON 배열로 바꾸면 간단한 설정값, 테스트 데이터, 샘플 입력을 빠르게 준비할 수 있습니다.

## FAQ
### 쉼표가 들어간 항목도 CSV로 바뀌나요?
네. 쉼표, 따옴표, 줄바꿈이 포함된 항목은 CSV 규칙에 맞춰 따옴표로 감싸고 내부 따옴표를 이스케이프합니다.

### 중복 항목은 어떻게 처리하나요?
`중복 제거` 옵션을 켜면 정리된 문구 기준으로 같은 항목을 한 번만 남깁니다. 원래 순서는 최대한 유지합니다.

### 민감한 목록을 넣어도 되나요?
도구가 서버로 전송하지는 않지만, 계정 정보나 개인정보처럼 민감한 내용은 가능하면 넣지 않는 편이 안전합니다.
