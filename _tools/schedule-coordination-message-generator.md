## 일정 조율 메시지 생성기가 왜 필요할까요?
일정 조율은 간단해 보여도 막상 보내려면 애매합니다.
너무 짧으면 무뚝뚝해 보이고, 너무 길면 핵심 시간이 묻히기 쉽습니다.
특히 처음 연락하는 상대, 팀 단체방, 인터뷰 일정처럼 **예의와 명확함을 동시에 챙겨야 하는 상황**에서는 문구를 한 번 더 다듬게 됩니다.

이 도구는 그런 망설임을 줄이기 위해 만들었습니다.
상황, 말투, 시간 후보, 응답 요청 시점을 고르면 **바로 복사해 보낼 수 있는 일정 조율 메시지 초안**을 만들어줍니다.

## 이런 상황에서 특히 유용해요
### 1) 회의 시간 제안
프로젝트 킥오프, 1:1 미팅, 외부 미팅처럼 참석자에게 가능한 시간을 물어봐야 할 때 빠르게 쓸 수 있습니다.

### 2) 약속 날짜 조율
지인과 식사, 병원 예약 재조정, 상담 일정 변경처럼 너무 딱딱하지 않게 시간을 제안하고 싶을 때 유용합니다.

### 3) 인터뷰 / 면담 안내
채용 인터뷰, 학생 면담, 상담 안내처럼 정중한 말투가 중요한 경우에도 안전한 기본 초안을 만들기 좋습니다.

## 어떻게 동작하나요?
1. 상황을 고릅니다. (회의, 약속, 인터뷰, 상담 등)
2. 채널과 말투를 고릅니다. (메신저 / 이메일, 보통 / 정중 / 캐주얼)
3. 상대 이름이나 호칭, 목적, 제안 시간을 입력합니다.
4. 답변을 받고 싶은 시점과 추가 메모가 있으면 넣습니다.
5. 생성 버튼을 누르면 짧은형, 기본형, 정중형 메시지가 함께 만들어집니다.

## 사용 방법
1. 상대 호칭과 일정 목적을 입력합니다.
2. 제안할 시간 후보를 한 줄에 하나씩 입력합니다.
3. 상황, 채널, 말투, 응답 요청 여부를 선택합니다.
4. `메시지 생성`을 누릅니다.
5. 결과 중 가장 자연스러운 버전을 골라 복사해 전송합니다.

## 사용 예시
### 예시 1) 팀 회의 시간 조율
- 상황: 회의
- 채널: 슬랙/카톡
- 말투: 보통
- 제안 시간:
  - 화요일 오후 2시
  - 수요일 오전 10시
  - 목요일 오후 4시

→ 단체 채팅에 바로 붙여넣기 쉬운 일정 조율 문구를 빠르게 만들 수 있습니다.

### 예시 2) 인터뷰 일정 안내 메일
- 상황: 인터뷰
- 채널: 이메일
- 말투: 정중
- 응답 요청: 내일 오후 3시 전

→ 지나치게 장황하지 않으면서도 예의를 갖춘 초안을 만들 수 있습니다.

## FAQ
### 여러 시간 후보를 한 번에 넣어도 되나요?
네. 줄바꿈으로 구분해 넣으면 번호 목록처럼 보기 좋게 정리해줍니다.

### 상대 이름 없이도 쓸 수 있나요?
가능합니다. 이름을 비워두면 “안녕하세요” 같은 일반형 문구로 생성됩니다.

### 결과를 그대로 보내도 되나요?
대부분의 상황에서 바로 쓸 수 있지만, 중요한 외부 일정이나 채용 관련 연락이라면 직함, 장소, 회의 링크 같은 세부 정보는 마지막에 한 번 더 확인하는 것이 좋습니다.
