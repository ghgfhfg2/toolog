## 유튜브 이미지 세트 메이커가 필요한 이유
유튜브 업로드 시 썸네일, 쇼츠 커버, 채널 배너 등 규격이 제각각이라 매번 수동 편집이 필요합니다.
이 도구는 **이미지 1장으로 업로드용 세트를 자동 생성**해 제작 시간을 크게 줄입니다.

## 생성되는 출력 규격
- 유튜브 썸네일: 1280×720
- 쇼츠 커버: 1080×1920
- 채널 배너: 2560×1440
- 채널 아이콘: 800×800
- 워터마크: 150×150

## 사용 방법
1. 기준 이미지를 업로드합니다.
2. 프레이밍 모드(cover/contain)와 배경색을 선택합니다.
3. `유튜브 세트 생성`을 누르면 출력 카드가 생성됩니다.
4. 개별 다운로드 또는 `전체 다운로드`를 사용합니다.

## 실전 팁
- 인물 중심 이미지는 `cover`가 썸네일 가시성에 유리합니다.
- 로고/텍스트 중심 이미지는 `contain`으로 잘림을 방지하세요.
- 생성 후 [PNG 압축기]({{ '/tools/png-compressor/' | relative_url }})로 용량까지 최적화하면 업로드 효율이 올라갑니다.

## 요약
유튜브 이미지 세트 메이커는 채널 운영에 필요한 이미지 규격 작업을 자동화해,
업로드 준비 시간을 줄이고 결과물 일관성을 높여줍니다.
