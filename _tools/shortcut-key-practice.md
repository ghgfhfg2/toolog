## 왜 단축키 기억 퀴즈 연습기가 필요할까요?
자주 쓰는 단축키는 한 번 외우면 업무 속도를 크게 줄여주지만, 막상 필요할 때는 마우스부터 손이 가기 쉽습니다. 특히 브라우저 탭 이동, 문서 편집, 화면 전환처럼 매일 쓰는 동작은 짧게 반복 연습할수록 기억에 남습니다.

이 도구는 운영체제와 연습 범위를 고른 뒤 **동작 설명을 보고 알맞은 단축키를 고르는 learning형 퀴즈 도구**입니다. 틀린 문제는 오답 목록에 남겨 다시 복습할 수 있어, 단순 목록을 읽는 것보다 기억을 확인하기 좋습니다.

## 어떻게 쓰나요?
1. 맥, 윈도우, 공통 단축키 중 연습할 환경을 고릅니다.
2. 브라우저, 문서 편집, 시스템, 전체 범위 중 하나를 선택합니다.
3. `새 문제`를 눌러 동작 설명을 확인합니다.
4. 보기 중 맞는 단축키를 고르고 해설을 읽습니다.
5. 틀린 문제는 `오답 복습`으로 다시 풀어봅니다.

## 이런 상황에서 특히 유용해요
### 1) 업무 시작 전 3분 워밍업
메일과 문서를 열기 전 몇 문제만 풀어도 오늘 자주 쓸 단축키를 다시 떠올릴 수 있습니다.

### 2) 맥과 윈도우를 번갈아 쓸 때
운영체제별로 Ctrl, Command, Option, Alt 조합이 헷갈릴 때 필요한 환경만 골라 연습할 수 있습니다.

### 3) 팀 온보딩이나 신입 교육 자료를 만들 때
단축키 목록을 전달하는 대신 퀴즈로 확인하면 실제 기억 여부를 더 쉽게 점검할 수 있습니다.

## FAQ
### 단축키가 앱마다 다르면 어떻게 하나요?
이 도구는 브라우저와 일반 문서 편집에서 많이 쓰는 기본 단축키를 중심으로 구성했습니다. 특정 앱의 전용 단축키는 해당 앱 도움말을 함께 확인하는 것이 좋습니다.

### 키를 직접 누르는 방식이 아닌 이유는 무엇인가요?
브라우저에서 실제 단축키 입력을 감지하면 새 탭 열기, 페이지 이동 같은 기본 동작과 충돌할 수 있습니다. 그래서 안전한 4지선다 퀴즈 방식으로 만들었습니다.

### 오답만 따로 볼 수 있나요?
네. 틀린 문제는 세션 안에서 오답 목록에 쌓이며, 오답 복습 버튼으로 다시 풀 수 있습니다. 새로고침하면 초기화됩니다.

## 요약
단축키 기억 퀴즈 연습기는 **브라우저·문서·시스템 기본 단축키를 짧은 문제로 반복하는 learning형 도구**입니다. 직전 checker와 최근 생활 준비형 흐름에서 벗어나 업무 학습 맥락을 보강해 툴 포맷과 사용 상황의 다양성을 넓힙니다.
