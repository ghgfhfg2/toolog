## 대소문자 변환기가 필요한 상황
문서, 코드, 데이터 라벨 작업에서는 케이스 규칙이 통일되지 않으면 품질이 크게 떨어집니다.
이 도구는 다양한 케이스를 즉시 변환해 포맷 정리 시간을 줄여줍니다.

## 지원 형식
- UPPER
- lower
- Title Case
- camelCase
- snake_case

## 사용 예시
### 1) API 필드명 정리
문서에 있는 문장을 `camelCase` 또는 `snake_case`로 통일할 때 활용합니다.

### 2) 콘텐츠 제목 정리
대문자/소문자 규칙이 섞인 제목을 빠르게 표준화할 수 있습니다.

### 3) 데이터 전처리
CSV/시트 컬럼명을 일정한 규칙으로 맞출 때 유용합니다.

## 가독성 팁
변환 후에는 사람이 읽기 쉬운지 한 번 더 확인하세요.
형식이 맞아도 의미가 불명확하면 유지보수성이 떨어질 수 있습니다.
