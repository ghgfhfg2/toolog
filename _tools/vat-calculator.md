## 부가세 계산기, 검색 의도에 맞춘 설명
**부가세 계산기**는 `공급가 계산`, `합계 부가세 역산`처럼 실무형 금액 검증이 필요한 검색 의도에 최적화된 페이지입니다.

## 지원 계산 방식
- 공급가 기준: 공급가 입력 → VAT/합계 계산
- 합계 기준: 총액 입력 → 공급가/VAT 역산

## 사용 방법
### 1) 기준 선택
공급가 기준인지 합계 기준인지 먼저 선택합니다.

### 2) 금액 입력
원 단위 금액을 입력하면 결과가 자동 갱신됩니다.

### 3) 결과 검토
공급가/부가세/합계 값을 확인 후 문서에 반영합니다.

## 실전 예시
- 견적서 작성 시 세전/세후 금액 동시 확인
- 결제 내역 정리 시 VAT 분리
- 거래처와 금액 근거 공유

## FAQ
### 공급가 기준과 합계 기준은 뭐가 다른가요?
기준이 되는 입력값이 다릅니다. 공급가 입력인지 총액 입력인지 먼저 확인하세요.

### 세율 10% 외 계산도 가능한가요?
현재 페이지는 10% 기준입니다. 다세율 지원이 필요하면 확장 버전을 추가하는 게 좋습니다.

### 소수점 처리는 어떻게 하나요?
정산 기준을 사내 규칙으로 통일해 사용하세요.

## 요약
부가세 계산기는 단순 계산이 아니라,
견적/정산/증빙 과정의 오류를 줄이는 업무 안정화 도구입니다.
