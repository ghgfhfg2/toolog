## 부가세 계산기, 검색 의도에 맞춘 설명
**부가세 계산기**는 공급가에서 VAT와 합계를 계산하거나, 부가세 포함 합계에서 공급가와 VAT를 역산하는 무료 계산 도구입니다. 원 단위 반올림·버림·올림 기준을 직접 선택할 수 있어 견적서와 정산 금액 검산에 유용합니다.

## 지원 계산 방식
- 공급가 기준: 공급가 입력 → VAT/합계 계산
- 합계 기준: 총액 입력 → 공급가/VAT 역산
- 원 단위 처리: 반올림/버림/올림 선택
- 결과 복사: 공급가·부가세·합계 요약 복사

## 사용 방법
### 1) 기준 선택
공급가 기준인지 합계 기준인지 먼저 선택합니다.

### 2) 금액 입력
0원 이상의 정수 금액을 입력하면 결과가 자동 갱신됩니다. 합계 역산 시에는 부가세 포함 총액을 입력하세요.

### 3) 원 단위 처리와 결과 검토
정산 정책에 맞춰 반올림·버림·올림을 선택한 뒤 공급가/부가세/합계를 확인하세요. 합계 기준에서는 공급가를 먼저 처리하고 남는 금액을 부가세로 계산해 입력한 총액을 유지합니다.

## 실전 예시
- 견적서 작성 시 세전/세후 금액 동시 확인
- 결제 내역 정리 시 VAT 분리
- 거래처와 금액 근거 공유

## FAQ
### 공급가 기준과 합계 기준은 뭐가 다른가요?
기준이 되는 입력값이 다릅니다. 공급가 입력인지 총액 입력인지 먼저 확인하세요.

### 세율 10% 외 계산도 가능한가요?
현재 페이지는 국내 일반 과세에서 자주 쓰는 10% 기준으로만 계산합니다. 영세율·면세 여부나 다른 세율은 거래 조건과 증빙을 별도로 확인하세요.

### 소수점 처리는 어떻게 하나요?
원 단위 반올림, 버림, 올림 중 정산 기준에 맞는 방식을 선택하세요. 세금계산서나 계약서 작성 전에는 거래처의 처리 규칙과 최종 금액을 다시 확인하는 것이 좋습니다.

## 요약
부가세 계산기는 공급가·VAT·합계를 빠르게 나누고 원 단위 처리 차이를 확인해 견적/정산/증빙 과정의 금액 오류를 줄이는 도구입니다.
