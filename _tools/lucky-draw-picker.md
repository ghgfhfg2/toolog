## 랜덤 추첨기, 이런 상황에서 바로 써보세요
- 이벤트 댓글/신청자 중 **당첨자 뽑기**
- 스터디·모임에서 **발표 순서 랜덤 결정**
- 팀 프로젝트에서 **무작위 팀 배정**

명단만 붙여넣고 당첨 인원만 정하면 바로 결과가 나옵니다.

## 사용 방법
1. 참여자 이름을 한 줄에 한 명씩 입력
2. 당첨 인원 입력
3. **추첨하기** 버튼 클릭

결과는 번호 순서로 출력되며, **결과 복사** 버튼으로 바로 공유할 수 있습니다.

## 추첨 규칙
- 앞뒤 공백 제거
- 빈 줄 제외
- 중복 이름 제거
- 당첨 인원은 `1 ~ 참여자 수` 범위로 자동 보정

## FAQ
### 참여자 순서가 당첨 확률에 영향을 주나요?
아니요. 입력 순서와 상관없이 동일 확률로 추첨합니다.

### 모바일에서도 동작하나요?
네. 브라우저에서 작동하므로 모바일/PC 모두 사용 가능합니다.

### 서버로 데이터가 전송되나요?
아니요. 계산은 브라우저에서만 처리되어 명단이 외부로 전송되지 않습니다.
