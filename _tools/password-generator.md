## 비밀번호 생성기, 왜 필요한가요?
같은 비밀번호 재사용이나 짧은 패스워드는 계정 탈취 위험을 크게 높입니다.  
이 도구는 **강력하고 예측 어려운 랜덤 비밀번호**를 빠르게 만들어, 회원가입/계정 변경 시 바로 쓸 수 있게 도와줍니다.

## 핵심 기능
- **길이(4~128자) + 생성 개수(1~20개) 설정**
- **대문자/소문자/숫자/특수문자 선택 조합**
- **헷갈리는 문자 제외 옵션** (`O/0`, `l/I/1`, `B/8`, `S/5`, `Z/2` 등)
- **강도 추정 + 문자풀 + 엔트로피(비트) + 가능 조합수 표시**
- **대량 생성 시 중복 최소화(가능한 범위 내 고유 비밀번호 우선 생성) + 전체 복사 지원**

## 사용 방법
1. 비밀번호 길이와 개수를 입력합니다.
2. 포함할 문자 유형을 선택합니다.
3. 필요하면 `헷갈리는 문자 제외`를 켭니다.
4. `비밀번호 생성`을 누르고 결과를 복사해 사용합니다.

## 보안 팁
- 서비스마다 **서로 다른 비밀번호**를 사용하세요.
- 가능하면 **비밀번호 관리자**에 저장해 관리하세요.
- 2단계 인증(2FA)을 함께 설정하면 보안이 크게 강화됩니다.

## FAQ
### 생성된 비밀번호는 안전한가요?
브라우저의 암호학적 난수(`crypto.getRandomValues`)를 사용해 생성합니다.

### 특수문자를 빼고 생성해도 되나요?
가능하지만, 문자 풀이 줄어들어 예측 난이도가 낮아질 수 있습니다.

### 강도 추정은 절대적인가요?
아니요. 길이/문자풀 기반의 참고 지표이며, 실제 보안성은 비밀번호 재사용 여부와 2FA 설정 여부에도 크게 좌우됩니다.

## 요약
비밀번호 생성기는 계정 보안을 위한 기본 도구입니다.  
**길이·문자 조합·가독성 옵션**을 조절해 실사용 가능한 안전한 비밀번호를 빠르게 만들 수 있습니다.
