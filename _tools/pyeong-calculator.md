## 평수 계산기, 왜 자주 필요할까요?
부동산 플랫폼은 ㎡ 중심, 대화는 평 중심인 경우가 많습니다.  
이 도구는 **㎡↔평 변환**뿐 아니라 **평당가 계산**까지 한 번에 처리해 매물 비교 시간을 줄여줍니다.

## 핵심 기능
- **양방향 변환**: ㎡ 입력 시 평 자동 계산, 평 입력 시 ㎡ 자동 계산
- **평당가 계산**: 매매가/전세가를 입력하면 평당가 즉시 산출
- **단위 보조 표시**: 원·만원·억원 단위를 함께 제공해 해석이 쉬움
- **입력 검증**: 0 이하 값이나 비정상 입력 시 안내 메시지 제공
- **모바일 대응**: 현장에서도 빠르게 면적/가격 비교 가능

## 사용 방법
### 1) 면적 입력
`㎡` 또는 `평` 중 익숙한 단위 하나만 입력하면 반대 단위가 자동 계산됩니다.

### 2) 금액 입력(선택)
매물 총액(원)을 입력하면 평당가가 바로 표시됩니다.

### 3) 결과 해석
- 변환 면적(㎡/평)
- 총액 기준 평당가
- 보조 단위(만원·억원)
를 함께 확인해 매물을 비교하면 됩니다.

## 실전 예시
### 아파트 매물 면적 비교
84㎡는 약 25.41평으로 계산됩니다. 서로 다른 표기 매물을 같은 기준으로 볼 수 있습니다.

### 평당가 빠른 판단
매매가 9억, 전용 84㎡(약 25.41평)라면 평당가는 약 3,541만원으로 확인됩니다.

### 분양 정보 해석
분양 공고의 공급면적(㎡)을 평으로 바꿔 체감 면적을 빠르게 파악할 수 있습니다.

## FAQ
### 1평은 정확히 몇 ㎡인가요?
1평 = 3.305785㎡ 기준으로 계산합니다.

### 전용/공급 중 무엇을 넣어야 하나요?
실사용 비교는 전용면적, 분양/공시 기준 비교는 공급면적을 주로 사용합니다.

### 평당가는 어떻게 계산하나요?
총액(원) ÷ 면적(평)으로 계산합니다.

## 요약

**면적 비교 + 평당가 판단을 동시에 지원하는 부동산 실전 도구**입니다.
