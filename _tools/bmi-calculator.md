## BMI를 10초 만에 확인하세요
다이어트/증량 계획 전에 가장 먼저 보는 지표가 BMI(체질량지수)입니다.  
이 도구는 **키와 몸무게만 입력하면 즉시 BMI와 체중 분류**를 보여주고, **정상 체중 범위**까지 같이 계산합니다.

## 핵심 기능
- **BMI 실시간 계산** (입력 즉시 반영)
- **체중 분류 자동 판정** (저체중/정상/과체중/비만/고도비만)
- **키 기반 정상 체중 범위 제공**
- **선택 입력 시 BMR(기초대사량) 계산**
- **모바일 입력 최적화 + 결과 복사**

## 사용 방법
### 1) 키(cm) 입력
예: `170`

### 2) 몸무게(kg) 입력
예: `65`

### 3) (선택) 나이/성별 입력
입력하면 BMR도 함께 계산됩니다.

## 계산 예시
- 키: 170cm
- 몸무게: 65kg

BMI는 약 **22.49**로 계산되며, 일반적으로 **정상 범위**에 해당합니다.  
같은 키 기준으로 정상 체중 범위도 함께 확인할 수 있어 감량/증량 목표 설정에 유용합니다.

## 주의 사항
- BMI는 **성인 기준 참고 지표**입니다.
- 운동선수처럼 근육량이 높은 경우 실제 체지방 상태와 차이가 날 수 있습니다.
- 질환 진단/치료 목적이라면 의료진 상담을 권장합니다.
