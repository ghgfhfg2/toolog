## 예금 이자 계산기, 언제 유용할까요?
정기예금/파킹통장/단기 예치 상품을 비교할 때는 **세전 금리만 보면 체감 수익이 다르게** 느껴질 수 있습니다.  
이 도구는 원금·금리·기간·과세 조건을 넣으면 세후 기준 만기금액까지 한 번에 보여줍니다.

## 핵심 기능
- **단리 / 월복리 방식 선택**
- **과세 15.4% / 비과세 0% 즉시 비교**
- **세전 이자·세금·세후 이자·만기금액 동시 계산**
- **결과 복사 버튼 제공**

## 사용 방법
1. 예치금(원), 연이율(%), 예치기간(개월)을 입력합니다.
2. 이자 계산 방식(단리/월복리)을 선택합니다.
3. 과세 유형(15.4%/0%)을 선택합니다.
4. 세전·세후 결과를 확인하고, 필요하면 복사해 상품 비교표에 붙여넣으세요.
