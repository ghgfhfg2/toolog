## 팁 계산기, 왜 필요할까요?
해외 여행이나 외국인 고객 응대, 글로벌 결제 환경에서는 팁 계산이 자주 필요합니다.  
이 도구는 **팁 금액·총 결제금액·1인당 분할 금액**을 한 번에 보여줘 빠르게 결제 결정을 도와줍니다.

## 핵심 기능
- **퍼센트/직접 입력 2가지 팁 모드**
- **세금/서비스요금 포함 기준 선택**
- **인원수 분할(더치페이) 자동 계산**
- **실시간 계산 + 복사 버튼 지원**

## 사용 방법
1. 결제 금액(기본 금액, 세금/서비스요금)을 입력합니다.
2. 팁 계산 모드를 선택합니다. (비율 또는 직접 금액)
3. 인원수를 입력하면 1인당 금액이 자동 계산됩니다.
4. 결과를 복사해 정산 메시지로 바로 공유할 수 있습니다.

## FAQ
### 팁 비율과 직접 입력 금액 중 무엇을 쓰면 되나요?
둘 다 지원합니다. 비율 기준이면 팁 % 모드, 금액이 이미 정해졌다면 직접 입력 모드를 사용하세요.

### 세금/서비스요금을 포함한 금액 기준으로 계산할 수 있나요?
네. `팁 계산 기준`에서 포함 여부를 선택하면 자동 반영됩니다.

### 1인당 금액이 소수점으로 나오면?
1인당 값은 소수점 2자리까지 제공됩니다. 실제 결제에서는 반올림 정책에 맞춰 정산하세요.
