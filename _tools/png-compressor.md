## 문제
이미지 파일이 너무 크면 페이지 로딩이 느려지고, 업로드 시간도 길어집니다.
특히 블로그/쇼핑몰에서는 이미지 최적화가 체감 성능에 직접 영향을 줍니다.

## 설명
이 도구는 브라우저에서 이미지를 다시 인코딩해 용량을 줄입니다.

- **출력 포맷 선택 가능**: WebP(권장) / JPEG / PNG
- **품질 슬라이더 제공**: 품질과 용량을 트레이드오프하며 조정
- **비교 정보 제공**: 원본 vs 압축 결과 용량을 즉시 확인

> 업로드한 이미지는 브라우저 안에서 처리되며 서버로 전송되지 않습니다.

## 예시
- 블로그 본문 이미지 업로드 전 2~5MB 이미지를 200~800KB대로 축소
- 쇼핑몰 상세 이미지 로딩 속도 개선
- 메신저/커뮤니티 업로드 제한 용량 맞추기

## 요약
용량이 큰 이미지는 검색 순위와 사용자 이탈률 모두에 악영향을 줄 수 있습니다.
게시 전에 5초만 투자해 최적화하면 체감 성능이 크게 좋아집니다.
