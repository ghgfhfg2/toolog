## 문제
이미지 화질을 크게 해치지 않으면서 파일 용량을 줄여야 할 때가 많습니다.

## 설명
업로드한 PNG를 압축해 용량을 낮추고, 원본 대비 결과를 비교할 수 있도록 설계합니다.

## 예시
- 블로그 업로드 전 이미지 최적화
- 쇼핑몰 썸네일 용량 절감

## 요약
PNG 압축기는 페이지 로딩 속도와 업로드 효율을 높이는 기본 도구입니다.
