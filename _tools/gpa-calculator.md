## 학점 계산기, 이런 상황에서 바로 써요
학기 말이 되면 **이번 학기 평점이 얼마나 나오는지**, 
장학금 기준이나 성적 유지 조건을 넘는지 빨리 확인하고 싶어집니다.

이 도구는 과목별 학점과 등급만 입력하면
**가중 평균 GPA, 평가 반영 학점, 총 평점, P/제외 학점**을 즉시 계산해 줍니다.
4.5 학점 계산기, 4.3 GPA 계산기, 4.0 기준 평점 계산기를 따로 찾지 않아도 한 화면에서 기준을 바꿔 비교할 수 있습니다.

## 핵심 기능
- 과목별 학점 × 등급 기준 가중 평균 계산
- 4.5 / 4.3 / 4.0 스케일 선택 지원
- P 과목은 GPA 제외 학점으로 별도 집계
- 총 평점, 반영 학점, 제외 학점 한 번에 확인
- 학점만 입력하고 등급을 빠뜨린 행, 범위를 벗어난 학점 입력 경고
- 결과 복사와 예시 입력, 전체 초기화 지원

## 사용 방법
1. 학교 기준에 맞는 학점 스케일을 선택합니다.
2. 과목별 학점을 입력합니다.
3. 각 과목의 등급을 선택합니다.
4. P/F 과목은 `P`로 선택해 GPA 계산에서 제외합니다.
5. GPA, 총 평점, 반영 학점, P/제외 학점을 확인합니다.
6. 필요한 경우 결과를 복사해 성적 계획표나 메모에 붙여넣습니다.

## 계산 방식
학점 계산은 보통 아래 방식으로 진행됩니다.

- 각 과목의 **학점 × 등급 점수**를 계산
- 이를 모두 더해 **총 평점** 산출
- 총 평점을 **평가 반영 학점 합계**로 나눠 GPA 계산

예를 들어 3학점 A+ 과목과 2학점 B+ 과목이 있다면,
각 점수를 학점만큼 반영해 평균을 내는 **가중 평균** 방식입니다.

## FAQ
### P 과목은 왜 GPA에서 제외되나요?
대부분의 학교에서 P/F 과목은 이수 여부만 표시하고 평점 평균에는 넣지 않습니다. 그래서 별도 학점으로만 보여줍니다.

### 학교마다 등급 점수가 조금 다른데 괜찮나요?
네. 4.5, 4.3, 4.0 스케일 중 학교 제도에 맞는 값을 선택하면 됩니다. 다만 세부 규정은 학교 학사안내를 최종 기준으로 확인하는 게 가장 정확합니다.

### 재수강 과목은 어떻게 넣어야 하나요?
실제 성적표에 반영될 기준에 맞춰 현재 인정되는 성적만 넣는 쪽이 보통 더 정확합니다. 학교마다 재수강 반영 방식이 다를 수 있습니다.

### 학점 입력 범위는 어떻게 되나요?
각 과목은 0보다 크고 30 이하인 학점만 계산합니다. 잘못된 값이나 등급이 빠진 행은 경고로 표시해 계산 실수를 줄입니다.

### 목표 GPA까지 얼마나 더 필요한지도 알 수 있나요?
이번 학기 또는 과목별 GPA는 이 계산기로 확인하고, 누적 GPA 목표까지 남은 학기 평균을 보고 싶다면 [목표 학점 계산기]({{ '/tools/required-gpa-calculator/' | relative_url }})를 함께 쓰면 됩니다.
