## 암기 퀴즈 생성기가 필요한 이유
공부할 때 가장 비효율적인 순간 중 하나는 **읽기만 계속하고 실제로는 기억이 나는지 확인하지 않는 것**입니다.
노트나 단어장을 오래 읽어도 막상 시험처럼 질문을 받으면 떠오르지 않는 이유가 여기에 있습니다.

특히 이런 상황에서 답답해집니다.
- 단어장이나 요약 노트는 있는데 스스로 테스트하기 귀찮을 때
- 시험 범위를 정리했지만 문제 형태로 바꿔보지 않았을 때
- 플래시카드 앱을 새로 세팅하기엔 지금 당장 빠르게 확인하고 싶을 때
- 틀린 것만 다시 보고 싶은데 수기로 표시하기 번거로울 때

이 도구는 그런 흐름을 줄이기 위해 만들었습니다.
**질문-정답 목록을 붙여넣기만 하면, 즉시 랜덤 퀴즈 세트로 바꾸고 정답 보기·자기 채점·오답만 다시 보기까지 한 번에** 할 수 있습니다.

## 어떻게 동작하나요?
입력 방식은 단순합니다.

1. 한 줄에 한 카드씩 입력합니다.
2. 질문과 정답은 `=` 또는 `:` 또는 탭, 쉼표로 나눕니다.
3. `퀴즈 만들기`를 누르면 카드 수와 진행 상태가 준비됩니다.
4. `다음 문제`를 누르며 스스로 답을 떠올립니다.
5. `정답 보기`로 확인한 뒤 `맞았어요 / 틀렸어요`를 기록합니다.
6. 마지막에는 정답률과 틀린 카드만 다시 모아 볼 수 있습니다.

즉, 무거운 학습 앱이라기보다 **지금 가지고 있는 메모를 바로 테스트 모드로 바꾸는 가벼운 학습 도구**에 가깝습니다.

## 이런 상황에서 특히 유용해요
### 1) 시험 전 빠른 셀프 테스트
중간고사·기말고사·자격증 시험 전에는 시간을 많이 들여 세팅하기보다, 이미 정리한 범위를 빨리 문제화하는 것이 중요합니다.

### 2) 영어 단어 / 외국어 표현 암기
단어 = 뜻, 표현 = 번역, 질문 = 예문 해석처럼 한 줄씩 넣으면 바로 단어장 퀴즈처럼 활용할 수 있습니다.

### 3) 면접 질문 / 발표 준비
예상 질문과 핵심 답변을 넣어두면, 실제로 질문받는 흐름처럼 연습할 수 있습니다.

### 4) 팀 교육 / 상품 지식 복습
상품 특징, 정책, 용어 설명을 카드로 만들어 신규 교육이나 주간 복습 자료로 돌리기 좋습니다.

## 사용 방법
1. 입력창에 질문-정답 세트를 한 줄씩 붙여넣습니다.
2. 카드 순서를 섞을지 여부를 선택합니다.
3. `예시 넣기` 또는 `퀴즈 만들기`를 누릅니다.
4. 화면의 질문을 보고 먼저 머릿속으로 답합니다.
5. `정답 보기`를 눌러 확인합니다.
6. `맞았어요` 또는 `틀렸어요`를 누릅니다.
7. 마지막에 정답률과 오답 카드를 확인합니다.

## 사용 예시
### 예시 1) 영어 단어 암기
- apple = 사과
- borrow = 빌리다
- accurate = 정확한

→ 랜덤 순서로 단어가 나오고, 뜻을 떠올린 뒤 스스로 채점할 수 있습니다.

### 예시 2) 한국사 연표 복습
- 갑오개혁 = 1894년
- 을사늑약 = 1905년
- 3·1 운동 = 1919년

→ 연도나 사건을 뒤섞어 보며 기억이 정말 나는지 빠르게 테스트할 수 있습니다.

### 예시 3) 면접 예상 질문 정리
- 우리 회사에 지원한 이유는? = 제품 방향성과 역할 적합성 중심으로 1분 답변
- 최근 가장 어려웠던 프로젝트는? = 문제 상황, 해결 방식, 결과 순으로 답변

→ 읽는 준비가 아니라 말로 답하는 연습용으로 바꿔 쓸 수 있습니다.

## FAQ
### 카드 수 제한이 있나요?
브라우저에서 바로 처리하는 도구라 일반적인 시험 범위나 단어장 수준에서는 충분히 가볍게 작동합니다. 너무 긴 본문형 데이터를 한 번에 넣기보다는 핵심 질문 단위로 나누는 편이 좋습니다.

### 질문과 정답이 아주 길어도 되나요?
가능합니다. 다만 셀프 테스트 도구 특성상 한 카드에 정보가 너무 많으면 암기 효율이 떨어질 수 있어, 핵심 문장 위주로 잘게 나누는 편이 더 좋습니다.

### 개인정보나 사내 기밀 내용을 넣어도 되나요?
민감한 자료는 항상 주의해야 합니다. 가능하면 이름, 계정 정보, 비공개 정책처럼 외부 노출이 민감한 내용은 제거한 뒤 사용하는 편이 안전합니다.
