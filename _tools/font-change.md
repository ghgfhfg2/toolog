## 폰트 변환기가 필요한 이유
SNS 소개문, 채널명, 프로필 텍스트를 꾸밀 때 매번 다른 사이트를 옮겨 다니면 비효율적입니다.
이 도구는 **입력 즉시 변환 + 클릭 복사**로 가장 빠른 폰트 변환 흐름을 제공합니다.

## 핵심 기능
- 실시간 폰트 변환(버튼 없이 자동 갱신)
- 다양한 유니코드 스타일 미리보기
- 클릭 즉시 복사 + 토스트 피드백
- 모바일 터치 환경 최적화

## 사용 방법
1. 텍스트를 입력합니다.
2. 변환된 폰트 리스트를 확인합니다.
3. 원하는 스타일을 클릭해 복사합니다.

## 실전 활용
- 인스타/유튜브 소개 문구 스타일링
- 닉네임/채널명 후보 빠른 테스트
- 이벤트 문구 강조 텍스트 제작
