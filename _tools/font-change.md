## 폰트 변환기가 필요한 이유
SNS 소개문, 채널명, 프로필 텍스트를 꾸밀 때 매번 다른 사이트를 옮겨 다니면 비효율적입니다.
이 도구는 **입력 즉시 변환 + 클릭 복사**로 가장 빠른 폰트 변환 흐름을 제공합니다.

## 핵심 기능
- 실시간 폰트 변환(버튼 없이 자동 갱신)
- 한국어·영어 이름으로 원하는 스타일 검색
- 다양한 유니코드 스타일 미리보기
- 클릭 또는 키보드로 개별 복사, 표시된 스타일 일괄 복사
- 글자 수, 표시 스타일 수, 실제 변환 여부 요약
- 모바일 터치 환경 최적화

## 사용 방법
1. 텍스트를 입력합니다.
2. 스타일 이름을 검색하거나 즐겨찾기만 모아 확인합니다.
3. 마음에 드는 스타일을 클릭해 복사하거나, 후보를 한 번에 비교하려면 표시된 스타일 전체를 복사합니다.

> 유니코드 폰트는 실제 글꼴 파일을 바꾸는 방식이 아닙니다. 영문과 숫자를 비슷한 모양의 유니코드 문자로 치환하므로 한글은 대부분 그대로이며, 일부 서비스에서는 문자가 지원되지 않을 수 있습니다.

## 실전 활용
- 인스타/유튜브 소개 문구 스타일링
- 닉네임/채널명 후보 빠른 테스트
- 이벤트 문구 강조 텍스트 제작
- 영문·숫자 중심 문구가 어떤 플랫폼에서 잘 보이는지 후보 비교
