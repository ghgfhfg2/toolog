## 이미지 리사이저가 필요한 상황
이미지 업로드 시 규격이 맞지 않아 잘리거나 품질이 깨지는 경우가 많습니다.
이 도구는 **사진 크기 조절**, **이미지 사이즈 변경**, **WebP 변환**을 빠르게 처리해, 채널별 권장 사이즈에 맞게 바로 사용할 수 있게 도와줍니다.
이미지는 서버로 올리지 않고 브라우저 안에서 처리하므로 간단한 썸네일, SNS 공유 이미지, 블로그 대표 이미지 작업에 적합합니다.
가로·세로 픽셀을 직접 입력하거나 프리셋을 고른 뒤, 원본 전체를 살릴지 크롭할지 선택할 수 있습니다.

## 주요 기능
- **프리셋 제공**: 인스타 정사각(1080×1080), OG(1200×630), 블로그 썸네일 등
- **직접 입력 모드**: 원하는 width/height를 자유롭게 입력
- **비율 고정 옵션**: 원본 비율을 유지한 채 너비나 높이 입력
- **맞춤 방식 선택**: 전체 맞춤(여백), 꽉 채우기(크롭), 지정 크기 늘리기
- **출력 포맷 선택**: PNG, WebP, JPEG 저장과 품질 조절 지원
- **여백 색상 지정**: 전체 맞춤 여백과 JPEG 투명 영역 배경색 선택
- **오류 방어**: 빈 파일, 이미지가 아닌 파일, 과도한 해상도 입력을 안내
- **결과 정보 표시**: 원본/목표/출력 해상도와 용량 확인
- **모바일 안전성**: 큰 이미지와 과도한 출력 크기를 제한해 브라우저 멈춤을 줄임

## 사용 방법
### 1) 이미지 업로드
리사이즈할 JPG, PNG, WebP 이미지를 업로드하면 원본 해상도가 자동으로 입력됩니다.
이미지가 아닌 파일이나 지나치게 큰 이미지는 결과 영역에 이유를 표시합니다.

### 2) 프리셋 또는 직접 입력
용도에 맞는 프리셋을 선택하거나 직접 픽셀 값을 입력합니다. 너무 큰 값은 브라우저 메모리를 보호하기 위해 제한됩니다.

### 3) 맞춤 방식 선택
원본 전체를 살려야 하면 전체 맞춤, 규격을 꽉 채워야 하면 크롭, 비율보다 정확한 픽셀이 더 중요하면 늘리기를 고릅니다.

### 4) 출력 포맷과 품질 확인
투명 배경이 필요한 로고·아이콘은 PNG를, 일반 사진과 블로그 이미지는 WebP나 JPEG를 선택합니다.
JPEG는 투명 배경을 유지하지 못하므로 여백 색상을 함께 확인하세요.
용량을 줄이고 싶다면 WebP를 먼저 고르고, 호환성이 더 중요하면 JPEG를 고르는 방식이 실용적입니다.

### 5) 리사이즈 실행 후 다운로드
미리보기 캔버스와 결과 정보를 확인한 뒤 결과 파일을 다운로드합니다.

## 실전 팁
- 블로그 대표 이미지는 16:9 비율(예: 800×450)이 관리하기 쉽습니다.
- SNS 공유 카드(OG)는 1200×630을 먼저 맞추면 플랫폼 호환성이 좋습니다.
- 상품 사진처럼 가장자리가 잘리면 안 되는 이미지는 전체 맞춤을 사용하세요.
- 투명 PNG를 JPEG로 저장하면 투명 영역이 선택한 여백 색상으로 채워집니다.
- 모바일에서 큰 원본 사진을 다룰 때는 1200px 안팎으로 먼저 줄이면 처리 실패 가능성이 낮아집니다.
- 업로드 폼이 정사각형만 받는 서비스는 크롭보다 전체 맞춤을 먼저 시험하면 잘림을 피하기 쉽습니다.
- 고해상도 이미지는 먼저 리사이즈 후, [PNG 압축기]({{ '/tools/png-compressor/' | relative_url }})로 용량 최적화하면 더 효율적입니다.
- 유튜브용 썸네일·배너·아이콘 세트가 필요하면 [유튜브 이미지 세트 메이커]({{ '/tools/youtube-image-kit/' | relative_url }})도 함께 사용할 수 있습니다.
