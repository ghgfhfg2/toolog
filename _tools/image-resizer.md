## 이미지 리사이저가 필요한 상황
이미지 업로드 시 규격이 맞지 않아 잘리거나 품질이 깨지는 경우가 많습니다.
이 도구는 **사진 크기 조절**을 빠르게 처리해, 채널별 권장 사이즈에 맞게 바로 사용할 수 있게 도와줍니다.

## 주요 기능
- **프리셋 제공**: 인스타 정사각(1080×1080), OG(1200×630), 블로그 썸네일 등
- **직접 입력 모드**: 원하는 width/height를 자유롭게 입력
- **비율 고정 옵션**: 원본 비율을 유지한 채 사이즈 변경
- **결과 정보 표시**: 원본/결과 해상도와 용량 확인

## 사용 방법
### 1) 이미지 업로드
리사이즈할 이미지를 업로드하면 원본 해상도가 자동으로 입력됩니다.

### 2) 프리셋 또는 직접 입력
용도에 맞는 프리셋을 선택하거나 직접 픽셀 값을 입력합니다.

### 3) 리사이즈 실행 후 다운로드
미리보기 캔버스를 확인한 뒤 결과 파일을 다운로드합니다.

## 실전 팁
- 블로그 대표 이미지는 16:9 비율(예: 800×450)이 관리하기 쉽습니다.
- SNS 공유 카드(OG)는 1200×630을 먼저 맞추면 플랫폼 호환성이 좋습니다.
- 고해상도 이미지는 먼저 리사이즈 후, [PNG 압축기]({{ '/tools/png-compressor/' | relative_url }})로 용량 최적화하면 더 효율적입니다.

이미지 사이즈를 적절히 맞추면 불필요한 리소스 로딩을 줄일 수 있어 페이지 속도와 UX 개선에 도움이 됩니다.
