## 퇴근 시간 계산기, 언제 유용한가요?
출근은 했는데 **정확히 몇 시에 퇴근 가능한지** 헷갈릴 때가 많습니다.
특히 유연근무·교대근무·외근 일정이 섞이면 체감으로 계산하기 어렵습니다.

이 도구는 **출근 시각 + 실제 근무시간 + 휴게시간**만 입력하면
예상 퇴근 시각과 총 체류시간을 바로 보여줍니다.

## 핵심 기능
- 출근 시각(시:분) 입력
- 목표 근무시간(시간) + 휴게시간(분) 합산 계산
- 예상 퇴근 시각 자동 산출
- 총 체류시간(근무+휴게)과 다음날 여부 표시

## 사용 예시
- 출근 09:00
- 근무시간 8시간
- 휴게시간 60분

→ 예상 퇴근 시각: **18:00**

## FAQ
### 휴게시간이 여러 번이면?
점심+짧은 휴식처럼 나눠져 있어도 총합 분(min)으로 넣으면 됩니다.

### 실제 퇴근과 차이가 나는 경우는?
연장근무, 회의 지연, 추가 휴게 등 변수가 있으면 결과와 달라질 수 있습니다.

### 모바일에서도 사용 가능한가요?
네. 브라우저에서 바로 계산되며 별도 설치가 필요 없습니다.
