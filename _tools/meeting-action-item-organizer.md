## 회의 액션 아이템 정리기가 필요한 이유
회의는 끝났는데 정작 남는 건 뒤죽박죽 메모인 경우가 많습니다.
누가 무엇을 하기로 했는지, 지금 당장 실행할 일인지, 단순 공유 메모인지 구분되지 않으면 **회의는 했는데 후속 진행이 느려지는 문제**가 생깁니다.

특히 이런 상황에서 자주 막힙니다.
- 메모는 많은데 실제 액션 아이템만 다시 골라내야 할 때
- 결정사항과 확인 필요 항목이 섞여 있어 공유 문구를 다시 써야 할 때
- 슬랙, 노션, 메일에 붙여넣을 후속 정리본이 바로 필요할 때
- 담당자와 마감일 자리만 빠르게 채운 체크리스트가 필요할 때

이 도구는 그런 흐름을 줄이기 위해 만들었습니다.
**회의 메모를 붙여넣으면 실행 항목, 확인 필요 항목, 공유 메모를 나눠 바로 복사 가능한 후속 정리본**으로 만들어줍니다.

## 어떻게 동작하나요?
사용 방식은 단순합니다.

1. 회의 제목과 메모를 입력합니다.
2. 필요하면 담당자 자리표시자, 우선순위 태그, 마감일 태그를 포함할지 고릅니다.
3. `정리하기` 버튼을 누릅니다.
4. 메모 안의 문장을 기준으로 실행 항목, 확인 필요 항목, 공유 메모가 자동 구분됩니다.
5. 결과를 복사해 메신저, 노션, 문서, 티켓 시스템에 붙여넣습니다.

결과에는 보통 아래 항목이 포함됩니다.
- **실행 항목(Action Items)**
- **확인 필요(Check / Follow-up)**
- **공유 메모(Notes / Decisions)**
- **빠른 공유용 한 줄 요약**

## 이런 상황에서 특히 유용해요
### 1) 회의 직후 5분 안에 후속 정리본이 필요할 때
산발적인 메모를 그대로 공유하기보다, 실행 항목만 추려 체크리스트 형태로 만들 수 있습니다.

### 2) 슬랙이나 팀 채팅방에 후속조치를 정리해야 할 때
담당자 자리표시자와 마감일 형식을 포함해 바로 붙여넣을 수 있습니다.

### 3) 노션, 지라, 아사나 같은 도구에 태스크를 옮기기 전
실행할 일과 단순 메모를 분리해 이중 기록을 줄이는 데 도움이 됩니다.

## 사용 방법
1. 회의 제목을 입력합니다.
2. 메모를 한 줄에 한 항목씩 붙여넣습니다.
3. 담당자, 우선순위, 마감일 태그 포함 여부를 고릅니다.
4. `정리하기` 버튼을 누릅니다.
5. 결과를 검토하고 `결과 복사`로 바로 공유합니다.

## 사용 예시
### 예시 1) 제품 주간회의
- 메모: 랜딩 문구 수정, 배너 시안 금요일까지 확인, 결제 오류 재현 로그 공유

→ 실행 항목, 확인 필요 항목, 공유 메모가 나뉘어 팀 채팅에 바로 붙여넣기 쉬운 형태로 정리됩니다.

### 예시 2) 클라이언트 미팅 후 정리
- 메모: 제안서 수정본 발송, 견적 범위 재확인, 일정표는 내부 검토 후 회신

→ 외부 발송용이 아닌 내부 후속조치 메모를 빠르게 구조화할 수 있습니다.

### 예시 3) 스터디/작업 회고
- 메모: 발표 자료 보완, 참고 링크 공유, 다음 회차 주제 확정

→ 해야 할 일과 단순 공유 항목을 분리해 다음 회차 준비가 쉬워집니다.

## FAQ
### 메모가 완전한 문장이 아니어도 되나요?
네. 짧은 키워드형 메모나 불릿 목록도 정리할 수 있도록 구성했습니다. 다만 한 줄에 한 항목씩 넣을수록 결과가 더 깔끔합니다.

### 담당자 이름을 자동으로 넣어주나요?
현재 버전은 담당자 자리표시자 형식으로 정리합니다. 실제 이름은 팀 상황에 맞게 결과를 복사한 뒤 채워 넣는 방식이 가장 안전합니다.

### 결정사항과 요청사항을 완벽하게 구분하나요?
기본 키워드와 문장 패턴을 기준으로 빠르게 정리합니다. 중요한 외부 커뮤니케이션 전에는 최종 검토를 한 번 더 권장합니다.
