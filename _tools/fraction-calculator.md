## 분수 계산기, 이런 때 바로 써요
문제집을 풀거나 레시피 비율을 계산할 때,
분수끼리 더하고 빼는 건 물론이고 **최종 결과를 약분해서 다시 확인하는 과정**이 은근 번거롭습니다.

이 도구는 두 분수를 입력하면 사칙연산 결과를 계산하고,
**약분 결과 / 대분수 / 소수 / 백분율**까지 한 화면에서 바로 보여줍니다.

## 핵심 기능
- 분수 덧셈·뺄셈·곱셈·나눗셈 지원
- 결과 분수 자동 약분
- 대분수 형태 함께 표시
- 소수값과 백분율 동시 확인
- 결과 복사 버튼 제공

## 사용 방법
1. 첫 번째 분자의 값과 분모를 입력합니다.
2. 연산 기호를 선택합니다.
3. 두 번째 분자의 값과 분모를 입력합니다.
4. 약분 결과, 대분수, 소수, 백분율을 확인합니다.

## FAQ
### 결과가 왜 대분수로도 나오나요?
가분수를 바로 해석하기 어렵다면 정수부와 분수부를 나눈 대분수 형태가 더 직관적이기 때문입니다.

### 음수 분수도 계산할 수 있나요?
네. 분자에 음수를 넣으면 음수 분수 계산도 가능합니다.

### 백분율은 어디에 쓰면 좋나요?
시험 점수 비율, 혼합 비율, 할인율처럼 분수를 퍼센트로 바로 해석해야 할 때 편리합니다.
