## 평균 계산기, 이런 상황에서 유용해요
시험 점수, 운동 기록, 지출 내역처럼 숫자 목록이 있을 때
**평균만 보는 것보다 중앙값·최솟값·최댓값까지 함께** 확인하면 더 정확한 판단이 가능합니다.

이 도구는 숫자만 붙여 넣으면 핵심 지표를 즉시 계산해 줍니다.

## 핵심 기능
- 숫자 목록 자동 파싱 (쉼표·공백·줄바꿈 지원)
- 평균(Mean), 중앙값(Median), 합계(Sum) 동시 계산
- 최솟값/최댓값 및 개수 확인
- 결과 복사 버튼 제공

## 사용 방법
1. 숫자 목록을 입력합니다.
2. 입력 즉시 결과가 자동 계산됩니다.
3. 필요하면 `결과 복사` 버튼으로 요약값을 공유합니다.

## FAQ
### 숫자는 어떤 형식으로 입력하면 되나요?
쉼표, 공백, 줄바꿈 모두 지원합니다. 예를 들어 `12, 18, 30` 또는 줄별 입력 둘 다 가능합니다.

### 소수점과 음수도 계산되나요?
네. `-2.5`, `3.14`처럼 소수점/음수 입력도 정상 계산됩니다.

### 중앙값은 어떻게 계산되나요?
숫자를 정렬한 뒤 가운데 값을 사용합니다. 데이터 개수가 짝수면 가운데 두 값의 평균을 중앙값으로 계산합니다.
