## 점심 메뉴 추천기가 필요한 이유
메뉴를 고르는 일은 사소해 보이지만, 막상 배고플 때는 이상하게 더 어려워집니다.
특히 이런 순간에 시간이 계속 새죠.

- 혼자 빨리 먹고 넘어가야 할 때
- 팀 점심인데 모두가 “아무거나”라고 할 때
- 예산은 정해져 있는데 메뉴가 안 떠오를 때
- 국물/밥류 여부처럼 지금 당기는 조건이 분명할 때

이 도구는 그런 망설임을 줄이려고 만들었습니다.
**식사 상황 + 함께 먹는 사람 + 예산 + 지금 끌리는 느낌**만 고르면,
바로 후보 3개를 추려서 복사하기 좋은 형태로 보여줍니다.

## 어떻게 추천하나요?
추천 로직은 아주 복잡한 AI가 아니라, 실제 식사 결정에 필요한 조건을 가볍게 반영하는 방식입니다.

1. **식사 시간**: 점심 / 저녁 / 야식
2. **인원 성격**: 혼밥 / 2~3명 / 팀 점심
3. **예산 구간**: 가볍게 / 보통 / 여유 있게
4. **분위기**: 든든한 한 끼 / 가볍고 깔끔하게 / 맵고 자극적으로 / 빨리 먹기 / 대화하기 좋게
5. **선호 옵션**: 뜨끈한 메뉴 포함, 밥류 포함

이 조건에 맞는 메뉴를 우선 점수화해 상위 후보를 보여줍니다.
즉, 무작정 랜덤이 아니라 **“지금 상황에서 고르기 쉬운 메뉴”를 정리해주는 선택 보조기**에 가깝습니다.

## 이런 상황에서 특히 유용해요
### 1) 회사 점심 메뉴 정하기
팀 점심은 메뉴 자체보다 **결정 피로**가 더 큰 경우가 많습니다.
이럴 때 결과를 복사해서 단톡방에 바로 붙이면 논의 시간이 줄어듭니다.

### 2) 혼밥인데 오래 고민하기 싫을 때
예산과 기분만 정해두면, 혼자 먹기 편한 메뉴가 위로 올라옵니다.
특히 빠르게 먹고 일해야 할 점심 시간에 잘 맞습니다.

### 3) 저녁 메뉴가 애매할 때
점심만이 아니라 저녁/야식 상황도 지원하므로,
퇴근 후 “배는 고픈데 뭘 먹을지 모르겠다”는 순간에도 바로 쓸 수 있습니다.

## 사용 방법
1. 식사 상황을 고릅니다.
2. 함께 먹는 사람 유형을 고릅니다.
3. 예산을 선택합니다.
4. 지금 당기는 느낌을 고릅니다.
5. 뜨끈한 메뉴, 밥류 포함 여부를 선택합니다.
6. `메뉴 추천 받기`를 누르면 상위 후보 3개가 생성됩니다.
7. 결과를 복사해 메모나 메신저에 바로 붙여넣을 수 있습니다.

## 사용 예시
### 예시 1) 혼자 빠르게 점심
- 식사 상황: 점심
- 함께 먹는 사람: 혼밥
- 예산: 가볍게
- 분위기: 빨리 먹기

→ 햄버거, 도시락/덮밥, 분식 세트 같은 빠른 메뉴가 상단에 뜰 가능성이 큽니다.

### 예시 2) 팀 점심, 대화 가능한 메뉴 찾기
- 식사 상황: 점심
- 함께 먹는 사람: 팀 점심 / 여럿
- 예산: 보통
- 분위기: 대화하기 좋게

→ 초밥, 파스타, 마라탕처럼 같이 먹으며 이야기하기 쉬운 메뉴가 우선 추천됩니다.

### 예시 3) 저녁에 든든하고 뜨끈하게
- 식사 상황: 저녁
- 함께 먹는 사람: 2~3명
- 예산: 보통
- 분위기: 든든한 한 끼
- 뜨끈한 메뉴 포함: 켬

→ 국밥, 김치찌개, 칼국수, 쌀국수 같은 메뉴가 더 잘 맞습니다.

## FAQ
### 메뉴 수가 적게 보일 때도 있나요?
조건이 매우 좁으면 후보 풀이 줄어들 수 있습니다. 이 경우 도구가 일부 조건을 완화해 다시 추천합니다.

### 다이어트식 / 채식 같은 세부 필터도 있나요?
현재 버전은 빠른 선택에 집중한 가벼운 추천기입니다. 추후 세부 식단 필터 확장 여지가 있습니다.

### 팀 점심에서 바로 투표처럼 써도 되나요?
네. 추천 결과 3개를 그대로 복사해 메신저나 사내 채팅방에 붙여 넣고 고르면 편합니다.
