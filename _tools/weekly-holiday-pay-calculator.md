## 주휴수당, 헷갈릴 때 바로 계산하세요
아르바이트·시급제 근무에서 가장 많이 헷갈리는 항목이 **주휴수당**입니다.
이 계산기는 시급과 근무 패턴만 입력하면, **대상 여부 + 주/월 예상 금액**을 바로 확인할 수 있도록 만들었습니다.

## 핵심 기능
- **주휴수당 대상 여부 자동 판정** (주 15시간 기준)
- **주휴시간 자동 계산** (주 소정근로시간 ÷ 주 소정근무일수)
- **법정 최대 8시간 자동 반영**
- **주/월 환산 주휴수당 동시 표시**
- **결과 복사 + 초기화 버튼 제공**

## 사용 방법
1. 시급(원)을 입력합니다.
2. 주 소정근로시간을 입력합니다.
3. 주 소정근무일수(예: 주 5일)를 입력합니다.
4. 개근 여부를 선택하면 결과가 즉시 계산됩니다.

## 계산 예시
- 시급: 10,030원
- 주 소정근로시간: 20시간
- 주 소정근무일수: 5일
- 개근: 예

→ 주휴시간 4시간, 주휴수당 약 40,120원

## 주의사항
- 본 계산기는 **간편 추정용**입니다.
- 실제 지급 여부/금액은 근로계약서, 취업규칙, 출결기록, 급여정산 기준으로 확정됩니다.
- 휴게시간·연장/야간/휴일 가산수당은 별도 계산 대상입니다.
